Agent Scripts

Agent Scripts are native Swift dylibs that run in-process with full TCC permissions. They're the fastest, most powerful way to extend Agent!.

Where They Live

Scripts are stored in ~/Documents/AgentScript/agents/. Each script is a single .swift file compiled on demand into a dylib.

Entry Point

import Foundation

@_cdecl("script_main")
public func scriptMain() -> Int32 {
    let folder = ProcessInfo.processInfo.environment["AGENT_PROJECT_FOLDER"]
              ?? FileManager.default.currentDirectoryPath
    let args   = ProcessInfo.processInfo.environment["AGENT_SCRIPT_ARGS"] ?? ""
    print("Project: \(folder)")
    print("Args: \(args)")
    return 0
}

App Automation

  • ScriptingBridge (preferred): import MusicBridge, then SBApplication(bundleIdentifier:).
  • NSAppleScript (fallback): one-liners when ScriptingBridge is awkward.
  • 50+ bridges shipped: Safari, Mail, Pages, Numbers, Keynote, Finder, System Events, Music, Notes, Reminders, Calendar, Contacts, Photos, and more.

Running Scripts

  • From the LLM: agent_script(action:"run", name:"my-script").
  • From the UI: Tools → Agent Scripts.
  • Pass data via the arguments parameter — it becomes AGENT_SCRIPT_ARGS in the script.

Safety

  • Deleting a script always creates a .Trash backup (restorable).
  • Scripts inherit Agent!'s TCC — grant permissions once, reuse everywhere.
  • Full Swift standard library + Foundation + SwiftUI + AppKit available.
