Tools & Capabilities

File Tools

  • read_file, write_file, edit_file — read, create, and modify files
  • list_files, search_files — browse and search project contents
  • create_diff, apply_diff — multi-line edits with preview

Git

The git tool exposes status, diff, log, commit, branch, and patch operations. Agent! commits after each successful build when working on code.

Xcode

The xcode tool builds Xcode projects, runs targets, adds and removes files from the project structure, bumps version/build numbers, and performs code review on specific line ranges.

Shell

  • execute_agent_command — runs shell commands as the current user via the Launch Agent (primary)
  • execute_daemon_command — runs as root via the Launch Daemon (admin tasks only)
Some destructive commands (rm -rf /, dd of=/dev/disk*, fork bombs, etc.) are blocked before they reach the shell. This guardrail cannot be bypassed with sudo or eval.

Web

  • web — Safari automation: open URLs, click, type, read content
  • selenium — Chrome/Firefox automation with WebDriver
  • web_search — search the internet via Tavily
  • fetch — fetch and strip HTML from a URL

Automation

  • applescript_tool — NSAppleScript, in-process with TCC
  • javascript_tool — JavaScript for Automation (JXA)
  • accessibility — element-based UI automation (AXorcist)

Details on app automation →

Planning & Memory

  • plan_mode — multi-step plans with progress tracking
  • memory — persistent notes across conversations (global or per-project)
  • index — repo-map with symbols and docs

Sub-Agents

Agent! can spawn isolated sub-agents for parallel work via spawn_agent, and talk to them via tell_agent. Up to three run concurrently.
