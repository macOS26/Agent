Getting Started

First Launch

When you first launch Agent!, macOS will ask for several permissions. Grant each one to unlock the full capability surface:

  • Accessibility — required to click buttons, read UI elements, and automate other apps
  • Automation — required to send AppleScript commands to apps like Safari, Mail, Finder
  • Full Disk Access — optional, enables reading files outside the sandbox
  • Screen Recording — optional, required for screenshot-based tools
If you skip a permission, Agent! will prompt again the first time a tool that requires it is used. You can always change permissions later in System Settings → Privacy & Security.

Adding an API Key

Agent! supports multiple LLM providers. Open Settings (⌘,) and navigate to Providers:

  • Anthropic Claude — paste an API key from console.anthropic.com
  • OpenAI — paste an API key from platform.openai.com
  • Ollama — runs locally, no key required (install from ollama.com)
  • Apple Intelligence — built in on Apple Silicon with macOS 26+

Setting a Project Folder

Agent! works best when pointed at a specific project. Click the folder icon in the header or drag a folder into the sidebar to set the current project. All file operations and shell commands run relative to this folder.

Installing the Privileged Helper (optional)

For tasks that require root access (disk cloning, software installation, system configuration), install the helper daemon from Settings → Daemon. You'll be prompted once to approve it in System Settings.
