LLM Providers

Agent! supports multiple LLM backends. Configure them in Settings → Providers.

Claude (Anthropic)

  • Paste your API key from console.anthropic.com.
  • Streaming responses via SSE with native tool calling.
  • Default model: claude-opus-4 or claude-sonnet-4.

OpenAI-Compatible

  • Works with any endpoint that speaks the OpenAI Chat Completions API.
  • Includes OpenAI, Groq, Together, Fireworks, DeepSeek, xAI, and local servers.
  • Set the base URL and API key in Settings.

Ollama (Local)

  • Install Ollama and pull a model (e.g. ollama pull llama3.1).
  • Agent! auto-discovers models running at http://localhost:11434.
  • 100% offline — nothing leaves your Mac.

Apple Intelligence (Foundation Models)

  • Available on macOS 26+ with Apple Intelligence enabled.
  • On-device inference — private and free.
  • Smaller context window than Claude/GPT; best for quick tasks.

Fallback Chain

Set a priority order in Settings → Fallback Chain. Agent! will try providers in sequence if one fails or hits a rate limit.
