Privacy & Security

What Agent! Sends Over the Network

  • Only your prompts and tool results go to the LLM provider you configured.
  • Using Ollama or Apple Intelligence? Nothing leaves your Mac.
  • Agent! itself has no telemetry, analytics, or phone-home.

API Keys

  • Stored in the macOS Keychain, never in plain text.
  • Scoped per-provider; revoke anytime from the provider's console.

Privileged Helper (AgentHelper)

  • Installed via SMAppService — you approve once in System Settings.
  • Runs as root so Agent! can execute admin tasks (disk operations, installers).
  • Communicates over XPC with the .privileged option.
  • Remove anytime: System Settings → General → Login Items → Background.

TCC Permissions

Agent! may request:

  • Accessibility — for UI automation (clicking, typing into other apps).
  • Automation — per-app prompts when running AppleScript or ScriptingBridge.
  • Full Disk Access — only if a script reads protected folders.
  • Screen Recording — only if you ask for screenshots.

Grant or revoke any of these in System Settings → Privacy & Security.

Shell Safety Guardrails

Agent! refuses destructive shell commands at the tool layer — rm -rf /, dd of=/dev/disk*, fork bombs, raw filesystem writes, and more. The guard runs before XPC, so even a confused LLM can't destroy your machine.

Local Data

  • Conversations: ~/Documents/AgentScript/sessions/.
  • Per-project memory: <project>/.agent/memory/.
  • Global memory: ~/Documents/AgentScript/memory/.
  • All files are plain text — inspect or delete anytime.
