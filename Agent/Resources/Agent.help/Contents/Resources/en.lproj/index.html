Welcome to Agent!

Agent! is an autonomous macOS app powered by multiple LLM providers. It can execute commands, automate apps, browse the web, and write code on your behalf.

Quick Start

  1. Launch Agent! and grant Accessibility & Automation permissions when prompted.
  2. Add an API key in Settings → Providers (or use a local Ollama model).
  3. Type a task in the input field and press Return.
  4. Watch the Activity Log stream progress as Agent! works.
