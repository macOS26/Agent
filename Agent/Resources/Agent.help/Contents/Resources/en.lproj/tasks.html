Giving Agent! a Task

Writing a Good Prompt

Agent! is autonomous — it will pick tools, run commands, and iterate on its own. Write tasks the way you'd write a short instruction to a capable assistant:

  • Be specific about the goal, not the mechanics: "rename every .jpg in Downloads to lowercase" works better than "loop through files and call mv".
  • Name the target file or folder when possible: "edit ContentView.swift so the header is centered".
  • State constraints up front: "don't touch anything in the Tests folder".

Attaching Files & Screenshots

Drop files, folders, or images directly onto the input field. Images become attachments the model can see. Text files are read into context.

Interrupting a Task

Click the Stop button in the header, or press ⌘., to cancel the current task. Agent! will halt at the next tool boundary.

Follow-up Messages

Once a task finishes, type again in the input field to continue the conversation. Agent! remembers everything in the current tab.

Each tab is an isolated conversation with its own history, project folder, and provider. Open new tabs with ⌘T.

Task Modes

Tabs can run in different modes (set in Settings → Agent Options):

  • Full — every tool group is available
  • Coding — file, git, xcode, shell only
  • Automation — AppleScript, JavaScript, Accessibility
  • Web — Safari, Selenium, web search
  • Plan — no tools run; Agent! drafts a plan you approve
